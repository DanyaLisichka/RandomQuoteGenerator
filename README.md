## Описание программы
Random Quote Generator – Это GUI-приложение на Python (Tkinter), генерирующее случайные цитаты с возможностью фильтрации и сохранения истории.

### Возможности
- Генерация случайной цитаты
- Фильтрация по автору и теме
- Добавление новых цитат
- Сохранение истории в JSON
- Загрузка истории при запуске
